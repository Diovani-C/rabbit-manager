# Rabbit manager

## Objective

This is a way to challenge myself, the main objective is to learn how to manage
and build a project, the focus is to follow a pipeline and planning of an
application that is going to be deployed to the real world.
The objective of the app is to help a manager organize their company and day to
day tasks.
Technologies and practices I want to improve my knowledge with this project

- SQL database (Postgresql)
- Designing and running tests
- CI/CD
- Authentication and authorization
- SolidJs
- BunJs
- Real time collaboration
- SPA
- Data validation
- i18n
- Documentation
- Error handling

Expected app features:

- Authentication and authorization
- Roles and access permissions
- Text documents storage and management
- Files upload with directories and folders
- Tasks management, with recurrence, assignments, due date and subtasks
- Problems reports, converts to tasks, notify group of people, retrigger when
not solved after x amount of time
- Notification system with visibility tracking
